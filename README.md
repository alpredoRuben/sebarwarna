# START PROJECT
* Buka terminal atau command line 
* Ketikkan perintah :
```js 
git clone https://github.com/alpredoRuben/sebarwarna.git  

atau unduh aplikasi ini
```
* Masuk ke folder sebarwarna 
```js
    cd sebarwarna
```
* Install node modules
```js
    npm install, 
    
    atau
    
    yarn add
```

* Jalankan program
```js
    npm run start

    atau 

    npm start
```
